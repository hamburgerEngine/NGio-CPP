# NGio-CPP

* A C++ library for the Newgrounds API.

* This is a work in progress.

# Current Features

* Logging in
* Getting the player's username
* Encryption
* Medal unlocking
* Highscore saving/updating

# Planned Features

- [x] Medals
- [x] Highscores
