# NGio-CPP

* A C++ library for the Newgrounds API.

# Current Features

* AES128 and RC4 Encryption
* Logging in
* Getting the player's username
* Encryption
* Medal unlocking
* Highscore saving/updating

# Planned Features

- [x] Highscores
